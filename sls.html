Star Labs Social

A Group Blog For Space Enthusiasts

Project Github

Star Labs Social is a basic group blogging application themed for space enthusiasts. The application is built using the Django Framework, and the many built-in tools like its Object Model Mapper, Class Based Views. The front end takes advantage of Django's template tags in order to build an interactive multi page application.
